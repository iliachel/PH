# Сайт "Вопросы по философии"

Этот проект представляет собой удобный, мобильный веб-интерфейс для чтения вопросов из файла `Questions.txt`.

## Функции
- 📱 **Адаптивность:** Удобно читать на телефонах и планшетах.
- 🔍 **Поиск:** Быстрый поиск по заголовкам разделов.
- 🧭 **Навигация:** Боковое меню для быстрого переключения между вопросами.

## Как запустить на GitHub Pages

Чтобы ваш сайт стал доступен по ссылке в интернете, выполните следующие шаги:

1. **Зайдите в свой репозиторий** на GitHub.
2. Перейдите во вкладку **Settings** (Настройки) сверху.
3. В левом боковом меню выберите раздел **Pages**.
4. В разделе **Build and deployment** убедитесь, что:
   - Source: **Deploy from a branch**.
   - Branch: Выбрана ваша основная ветка (например, `main` или та, в которую вы объединили изменения) и пака **/(root)**.
5. Нажмите кнопку **Save** (если она активна).
6. Через 1-2 минуты GitHub предоставит вам ссылку вида `https://ваше-имя.github.io/название-репозитория/`.

## Как обновлять вопросы
Просто редактируйте файл `Questions.txt`. Сайт автоматически подхватит изменения.
- Используйте `#` для заголовка вопроса.
- Используйте `---` на отдельной строке, чтобы отделить один вопрос от другого.
